This session introduces the concepts of Messages and Events, through Tutorial videos you will be able to discover how these can be used to reduce coupling in your application.

We will also investigate the various Software licenses used in modern software development. You will gain an understanding of each license avaiable and how the restrictions of use apply to your own applications

Learning Objectives:
  • Identify the most popular software licenses
  • Outline the strengths and weaknesses of the various mainstream software licenses
  • Select the most appropriate software license for your own IP
  • Implement events and messages to reduce Coupling in your own games
