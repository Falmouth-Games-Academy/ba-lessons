This session will examine the role of collections in Object Orientated Programming languages. You will research the most common collections which can be used to build your games.

Learning Objectives:
  • Explain the collection classes in C#, C++ and Unreal Engine
  • Demonstrate your understanding of the Collection classes
  • Explain methods of managing your Game Logic using Static Classes or Singeltons
