This session introduces you to basic programming in Python, tinkering with text to make dynamic stories. Further to this, you will setup PyGame and experiment with rendering simple 2D graphics. This will lay the foundation for your forthcoming sessions on tinkering graphics.

Learning Objectives:
  • Use variables to program dynamic content
  • Setup a PyGame project in PyCharm
  • Recognize the structure and syntax associated with a PyGame project
