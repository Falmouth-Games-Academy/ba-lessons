In this session you will learn the basics of Object Orientated Programming. You will understand good coding practices and how OOP can support this. As a group, you will research existing game controllers to help inform your approach to the brief.

Learning Objectives:
  • Describe the key concepts of Object Orientated Programming
  • Demonstrate an understanding of Single Responsibility Principle
  • Describe existing alternative game controllers
