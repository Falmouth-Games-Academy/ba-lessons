In this session we will look at developing your Unity skills by exploring the key aspects of platform games, namely hazards, platforms and death. This session will also introduce the concept of version control for game development using the popular 'git' framework.

Learning Objectives:
  • Implement hazards, platforms and player death in a Unity platformer
  • Extend single screen 2D platform levels
  • Use Unity collision components to detect interaction between GameObjects
  • Use TortoiseGIT to manage project files
