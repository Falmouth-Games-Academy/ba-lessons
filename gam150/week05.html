This session will introduce 3D first person shooters (FPS) as game genre that we will implement for the 2nd module assignment and we will look at how levels can be created and imported into unity

Learning Objectives:
  • Evaluate different approaches for creating level content in Unity
  • Create level content using a range of approaches
  • Understand the peer review process
